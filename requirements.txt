# Exact pins for a known-good GRPO stack. The TRL GRPO API (and its hard
# dependency on vLLM at import time) moves across releases, so loose ranges can
# pull an incompatible combo (e.g. transformers 5.x against trl 0.17). These
# versions are mutually consistent and what src/ is written against.
#
# Platform note: vLLM has no Windows build, and trl>=0.16 imports vLLM at module
# load — so GRPO does NOT run on native Windows. Use Linux or WSL2 (your local
# GPU works there), or run it on Hugging Face Jobs (see README).
